Go: Resources for learning Golang

September 17, 2019

Golang Books

I have only recently started learning Go; and have fallen in love with the language since.

I wanted to write a quick post highlighting the main resources I have been using on my journey so far.

Get Programming with Go

The first book I bought on the subject was Get Programming with Go by Nathan Youngman.

I found this book an excellent introduction to all of the main topics you’re likely to encounter in Go as a beginner. I still find myself using it as a reference as its just so clearly explained and layed out.

Definitely a buy, for any new an aspiring Go programmer.
